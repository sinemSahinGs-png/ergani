# Ergani 1 No'lu Aile Sağlığı Merkezi

Production-ready Next.js website for the family health center.

## Stack

- Next.js (App Router) + TypeScript + Tailwind CSS
- Framer Motion + Lenis
- React Hook Form + Zod
- Lucide icons

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Media

Add your assets when ready:

- `/public/videos/asm-hero.mp4` (optional `/public/videos/asm-hero.webm`)
- `/public/images/asm-video-poster.jpg` (SVG placeholder exists)
- `/public/images/doctors/*.jpg`

Until then, the UI shows branded placeholders / doctor initials.

## Deploy

Deploy to Vercel. Optionally set `NEXT_PUBLIC_SITE_URL` for sitemap/OG absolute URLs.
